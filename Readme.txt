
##General Info

StringMarshallerApp - console app. This is simple console project for calling C++ declaration methods. This project demonstrate how to call C++ exported
methods from your C# application (especially string passing between C# to C++ and vice-versa).


##Technologies

.Net Framework, C++, C sharp


##Setup

To run this project, open its solution file into Visual studio and build the solution by right clicking on solution in solution explorer and selecting 
Build Solution.
